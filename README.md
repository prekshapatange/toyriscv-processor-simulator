# toyriscv-processor-simulator

This project implements a simplified processor simulator in Java, modeling core CPU components such as instruction parsing, execution, register operations, and memory.

## Features
- Instruction parsing and decoding
- ALU operations
- Register and memory simulation
- Step-by-step execution tracing

## Architecture
- CPU
- ALU
- Register File
- Memory Unit

## Concepts Covered
- Computer Architecture
- Instruction-level execution
- CPU design fundamentals
